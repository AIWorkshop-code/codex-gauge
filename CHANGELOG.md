# Changelog

## 0.5.4 — 2026-07-13

- 移除 5H 额度在小组件、托盘、通知、右键菜单和历史页中的显示。
- 小组件改为聚焦 7D 剩余额度与重置倒计时的轻量单卡布局。
- 新增更精简的 400 × 160 基准画布、柔和绿色光感、状态徽标和进度端点动效。
- 适配官方新的单周窗口数据结构：7D 额度改位于 `primary` 且 `secondary` 为空时，仍能根据 10080 分钟窗口正确识别。
- Windows 发布脚本固定生成 x64 安装包，与内置 Codex x64 sidecar 保持架构一致。

## 0.5.3 — 2026-07-12

- macOS 改为直接检查 GitHub Releases，不再依赖需要正式签名的静默自动更新。
- 发现新版时显示版本号与更新内容，并打开 GitHub 下载页及覆盖安装提示。
- 手动检查会明确提示当前已是最新版或 GitHub 暂时无法连接。
- 增加每日最多自动显示一次的会员服务公告，可关闭、可从右键菜单再次查看，并在用户点击后通过系统浏览器打开购买页面。

## 0.5.2 — 2026-07-12

- 增加“仅显示 5H”正方形紧凑模式。
- 右键菜单可以在完整模式与仅 5H 模式之间切换。
- 显示模式、窗口尺寸和位置会自动保存。
- 小、中、大尺寸预设同时适配两种布局。

## 0.5.1 — 2026-07-12

- 使用全新的简约清新应用图标。
- 增加完整的 macOS ICNS 与 Windows ICO 多尺寸资源。
- macOS 与 Windows 打包配置改为使用统一品牌图标。

## 0.5.0 — 2026-07-12

首个公开 Release。

### 新功能

- Windows 与 Apple Silicon macOS 桌面组件。
- 官方 Codex App Server 实时额度读取与本地日志安全降级。
- 5H / 7D 额度、明确单位的重置倒计时和自动刷新。
- 浅色、深色与自动系统主题。
- 额度变化动画、周期流光和减少动态效果支持。
- 额度不足与额度恢复系统通知。
- 数据来源、更新时间与诊断信息。
- 5H / 7D 倒计时切换、登录启动与托盘 / 菜单栏模式。
- 最近 30 天本地用量历史、消耗速度与趋势图。
- GitHub Releases 自动更新支持。
- 动态无障碍额度描述。

### 隐私

- 不读取浏览器 Cookie、ChatGPT 密码或认证令牌。
- 历史快照仅保存在本机，用户可随时清除。
